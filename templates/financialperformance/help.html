How to find out the meaning of variables from financial performance?

In determining the financial performance of a company, many variables are needed. The variables used include Cash, Inventory, Current Asset, Fixed Asset, Total Asset, Current Liabilities, Long Term Liabilities, Total Liabilities, Outstanding Share, Total Equity, Total Revenue, Gross Profit, Operating Profit or EBIT, Interest Expense, Net Profit, Earnings per Share, Dividend Payment, and Stock Price (Close Year).

  • So, what is Cash?
  • what is Inventory?
  • what is Current Asset?
  • … etc

In this website, you find out the meaning of each variable from financial performance by clicking the word. For example, you want to know the definition of the cash, so just click the word of “cash” and the definition will appear.

How to display analysis of financial performance from the name, code or sector of the company?

Here are some ways to display an analysis of financial performance from the name, code or sector of the company:

  1. Visit the Analysis Financial Performance page via the link http://cakedev.pythonanywhere.com/analysis/ or click the Analysis Financial Performance menu on the sidebar menu.
  2. In the search box, you can click on the box and choose which company to analyze from the list of companies. Remember, you can only choose one company.

    3. Another way: In the search box you can directly enter the keywords of a company, for example, a word of the name, code or sector of the company.

  3. To start analyzing, click on Analyze button.
  4. The result will be shown directly.
  5. To see the reason of the result (Good or Not good), you can click on the result label (Good or Not good).

How to display the comparison of financial performance from the name, code or sector of the company?

Here are some ways to display an analysis of financial performance from the name, code or sector of the company:

  1. Visit the Comparison Financial Performance page via the link http://cakedev.pythonanywhere.com/comparison/ or click the Comparison Financial Performance menu on the sidebar menu.
  2. In the search box, you can click on the box and choose which company to analyze from the list of companies.

    3. Another way: In the search box you can directly enter the keywords of a company, for example, a word of the name, code or sector of the company.

  3. If you want to add another company, repeat method number 2.

    4. Remember, you can only select four company (maximum).

  4. To see the result, you can click on Compare button
  5. The result will be shown below and will be displayed as a bar chart. Each chart has a threshold. Green chart means good, because the company’s ratio value is greater than the threshold value. Red chart means not good, because the company’s ratio value is lower than the threshold value.
  6. If you want to see the detail value of the company, you can click the selected company chart
